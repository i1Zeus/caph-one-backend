# دليل شجرة الحسابات المحاسبية - iZeus ERP

## 📊 نظرة عامة

تم تصميم شجرة الحسابات المحاسبية في iZeus ERP وفقاً لأفضل الممارسات المحاسبية الدولية مع مراعاة احتياجات الأعمال المحلية. يتبع النظام مبدأ القيد المزدوج (Double Entry) ويدعم العملات المتعددة.

## 🔢 نظام الترقيم

يستخدم النظام ترقيماً من 4 خانات مع التوزيع التالي:

- **1000-1999**: الأصول (Assets)
- **2000-2999**: الخصوم (Liabilities)
- **3000-3999**: حقوق الملكية (Equity)
- **4000-4999**: الإيرادات (Revenue)
- **5000-5999**: المصروفات (Expenses)

## 📁 شجرة الحسابات التفصيلية

      └─ لتسجيل المبالغ المستحقة من العملاء

## ⚙️ التكوينات التلقائية للفواتير

تم إعداد النظام للقيد التلقائي للفواتير كالتالي:

### فواتير المبيعات:

- **نقدي**: من حـ/ الصندوق (1100) إلى حـ/ إيرادات المبيعات (4100)
- **آجل**: من حـ/ العملاء (1200) إلى حـ/ إيرادات المبيعات (4100)

### فواتير المشتريات:

- **نقدي**: من حـ/ تكلفة البضاعة المباعة (5100) إلى حـ/ الصندوق (1100)
- **آجل**: من حـ/ تكلفة البضاعة المباعة (5100) إلى حـ/ الموردون (2100)

## 🚀 كيفية استخدام شجرة الحسابات

### 1. إدخال الحسابات إلى قاعدة البيانات:

```bash
npm run seed:accounts
```

### 2. عبر واجهة البرمجة (API):

- **GET** `/accounting/accounts` - عرض جميع الحسابات
- **POST** `/accounting/accounts` - إضافة حساب جديد
- **GET** `/accounting/accounts/:id` - عرض حساب محدد
- **PATCH** `/accounting/accounts/:id` - تعديل حساب
- **DELETE** `/accounting/accounts/:id` - حذف حساب

### 3. نموذج إضافة حساب:

```json
{
  "name": "1700 - استثمارات طويلة الأجل",
  "type": "ASSET",
  "isCash": false,
  "currencyId": 1
}
```

## 📋 أفضل الممارسات

1. **الحفاظ على التسلسل الرقمي**: استخدم أرقام متسلسلة عند إضافة حسابات جديدة
2. **التسمية الواضحة**: استخدم أسماء واضحة ووصفية للحسابات
3. **التصنيف الصحيح**: تأكد من اختيار النوع الصحيح للحساب (ASSET, LIABILITY, etc.)
4. **تحديد طبيعة النقدية**: حدد بدقة الحسابات النقدية (isCash = true)
5. **العملات المتعددة**: ربط كل حساب بالعملة المناسبة

## 🔒 الأمان والصلاحيات

- يتطلب إنشاء أو تعديل الحسابات صلاحية `accounts:create` أو `accounts:update`
- لا يمكن حذف الحسابات المرتبطة بمعاملات
- يتم تسجيل جميع التغييرات في سجل النشاطات

## 📞 الدعم الفني
